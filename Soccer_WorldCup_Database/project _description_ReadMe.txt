
Database used 		  - SQL plus(Oracle Database)
Programming Language used - Java with JDBC

There are 3 spool files which are recorded.
	1.one spool file contains the initial table creation and the loading of data.
	2.second spool file contains the 16 queries which are to be exceuted
	3.the last spool file contains the queries that deal with the constraints.
	
There is a file named queries which consists of all the queries.
There is a file named databaseconnection_jdbc to load the excel data into the database.
